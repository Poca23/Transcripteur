Activation du microphone

Pour utiliser le transcripteur, vous devez autoriser l'accès au microphone.

📱 Mobile : Appuyez sur "Autoriser" dans la popup
🍎 iOS : Vérifiez Réglages → Safari → Microphone
🤖 Android : Vérifiez Paramètres → Chrome → Autorisations
⚠️

Microphone non autorisé

Le microphone est nécessaire pour la transcription.

📱 Instructions selon votre appareil :

🍎 iPhone/iPad :
Réglages → Safari → Microphone → Autoriser
🤖 Android :
Paramètres → Apps → Chrome → Autorisations → Microphone
💻 Ordinateur :
Clic sur l'icône 🔒 dans la barre d'adresse
